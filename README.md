## 🎯 Project Overview

A **production-ready reinforcement learning environment** for training autonomous drones 
to conduct search and rescue operations in post-disaster urban environments.

### Key Highlights

- 🎮 **3 Difficulty Levels** with progressive complexity (Easy → Medium → Hard)
- 🤖 **5 Baseline Agents** for benchmarking and comparison
- 🧠 **OpenAI Integration** for LLM-based action selection
- 📊 **Deterministic Scoring** with comprehensive evaluation metrics
- 🔍 **Partial Observability** - Limited visibility and fog of war
- 🔋 **Resource Management** - Battery-constrained operations
- 🌍 **Dynamic Hazards** - Aftershocks, fires, unstable structures
- 🛡️ **Safety-Critical** - Multi-objective optimization (rescue, resources, avoid hazards)

## 📋 Overview

A realistic reinforcement learning environment for training autonomous drones to conduct search and rescue operations in post-disaster urban environments. Features:

- **3 Difficulty Levels**: Easy (5 victims), Medium (12 victims), Hard (25 victims)
- **Partial Observability**: Fog of war and limited visibility ranges
- **Dynamic Hazards**: Aftershocks, fires, unstable structures
- **Battery Management**: Limited power for extended missions
- **Multi-objective Optimization**: Rescue victims, deliver resources, avoid hazards
- **OpenAI Integration**: LLM-based action selection support
- **REST API**: FastAPI web interface for Hugging Face Spaces
